About Keep Search app

'Keep Search' is a Python-Flask web app that allows to extract Google Search Console queries and pages to a database.

It was developed by Jaime Marcos Hernandez, as part of his final project at Higher Diploma in Data Analytics in National College of Ireland.

You can find the full code in GitHub: See source code on Github

In order to export data from Google Search Console you will need:

  1. A Google Account (make sure to sign up with same email used for your Google Search Console account).
  2. A property or more in Google Search Console.
  3. Create a project in Google Cloud.
  4. Generate API keys for your project.

Step 1: Signup into your Google account

If you do not already have a Google account, sign up.

Step 2: Visit Google Cloud Platform

Visit console.developers.google.com and sign in with your Google account.

Step 3: Agree terms and coditions

Your wil be probably asked to to agree to the terms and conditions. Select Yes, and click on Agree and Continue..

Step 4: Create a new project

Click on the plus button to create a new project.

Step 5: Save the project

Once your project is generated, a notification will be generated to warn you in the top right corner.

Step 6: Select your project name

You need to make sure that your project is selected at the top left drop down menu.

Step 7: Find Search Console API

On the left menu, click on 'Library'. You need to look for the Search Console API in input box.

Step 8: Enable API

To access private data, Search Console API needs to be enabled.

Step 9: Create credentials

This will generate the credentials to authorize the use of the API.

Step 10: Configure consent

The consent is the screen that will appear to the user within the browser, asking for permission when the API is requested.

Step 11: Make sure to save the consent

Give it a name and save it.

Step 12: Create credentials

For this app, you need to select Other, and name the credentials.

Step 13: Copy client ID and API keys

Step 14: Save your credentials into Keep Search app

Once you log into the app, you will be required the credentials you generate in Google Cloud.

You are done!
